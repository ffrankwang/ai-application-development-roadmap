# Week 12：部署与作品整理

目标：

- 把项目整理成可展示、可部署的作品。

每日任务：

- Day 1：为 AI Chat 或 RAG 项目写 Dockerfile。
- Day 2：写 docker-compose.yml。
- Day 3：加入结构化日志和基础错误处理。
- Day 4：整理 README、架构图、接口文档。
- Day 5：完成最终演示和 3 个月复盘。

量化验收：

- `docker compose up` 可启动。
- `/health` 可访问。
- 至少 5 个接口测试通过。
- README 包含启动、接口、架构、测试样例。
- 至少 1 个项目可作为作品展示。

输出位置：

- 代码：`src/`
- 笔记：`notes/`
