# Week 09：Function Calling / Tool Calling

目标：

- 让模型根据问题选择并调用工具。

每日任务：

- Day 1：理解 tool schema，定义工具输入输出。
- Day 2：实现 `calculate` 工具。
- Day 3：实现 `get_current_time` 工具。
- Day 4：实现 `search_knowledge_base` 工具。
- Day 5：整理工具调用日志和 10 个测试问题。

量化验收：

- 至少 3 个工具。
- 至少 10 个测试问题。
- 能打印工具名、参数、返回值。
- 模型能根据问题选择合适工具。

输出位置：

- 代码：`src/`
- 笔记：`notes/`
