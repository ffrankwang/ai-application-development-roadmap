# Day 10：Prompt 模板库复盘指南

今天目标：

整理 Week 02 的 prompt、few-shot、JSON 输出练习，形成一个小型 Prompt 模板库。

## 任务要求

- 整理 5 个 system prompt。
- 整理 1-2 个 few-shot prompt。
- 整理 3 个 JSON 输出 prompt。
- 记录每个 prompt 的适用场景。
- 更新本周 README。

## 模板格式

建议每个 prompt 这样记录：

```text
名称：
场景：
Prompt：
输入样例：
输出样例：
注意事项：
```

## 自检清单

- 是否能复用？
- 是否有输入输出样例？
- 是否说明适用场景？
- 是否记录了失败情况？

## 完成标准

- 至少 10 个 prompt 测试样例。
- 至少 3 条 prompt 设计经验。
- README 包含本周成果和运行说明。
