# Week 02：Prompt 与结构化输出

目标：

- 让模型按指定格式返回。
- 处理 JSON 输出不稳定问题。

每日任务：

- Day 1：写 5 个 system prompt 模板。
- Day 2：写 few-shot 示例，让模型固定风格回答。
- Day 3：要求模型返回 JSON，完成 `json.loads()` 解析。
- Day 4：加入 JSON 解析异常处理和错误提示。
- Day 5：整理 Prompt 模板库和测试样例。

量化验收：

- 至少 5 个 prompt 模板。
- 至少 3 个结构化输出任务。
- 至少 10 个 prompt 测试输入。
- JSON 解析失败时程序不崩溃。

输出位置：

- 代码：`src/`
- 笔记：`notes/`
