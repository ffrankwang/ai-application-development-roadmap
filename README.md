# AI Application Development Roadmap

这是我的 3 个月 AI 应用开发学习与项目输出仓库。

背景：

- 已有 Java 后端开发基础。
- 已完成 Python 基础语法复习。
- 当前目标是转向 AI 应用开发实战。

时间安排：

- 总周期：3 个月。
- 每周 5 天。
- 每天 2-3 小时。
- 每周一个可运行产出。

## 阶段路线

| 阶段 | 周期 | 主题 | 阶段产出 |
| --- | --- | --- | --- |
| Phase 01 | Week 01-04 | LLM API + FastAPI AI Chat | AI Chat 后端 |
| Phase 02 | Week 05-08 | RAG 企业知识库 | RAG 知识库问答系统 |
| Phase 03 | Week 09-12 | Agent + 生产化 | AI SQL/日志分析助手 + Docker 部署 |

## 每周输出要求

每周目录至少包含：

- `README.md`：本周目标、任务、验收标准。
- `src/`：代码输出。
- `notes/`：学习笔记、踩坑记录、接口测试记录。

每周完成标准：

- 至少 1 个可运行脚本或服务。
- 至少 3 个测试样例。
- README 更新运行命令和完成情况。
- 能用自己的话讲清楚本周核心链路。
